# Contributing

1. **Issues** — [GitHub Issues](https://github.com/OrithmicSoftware/terminal-hs-game/issues) for bugs and small feature ideas.
2. **Branches** — use short-lived branches off `main`; open a PR with a clear description.
3. **CLI** — `node game.mjs` for the Node build; keep `src/` changes consistent with existing patterns.
4. **Web** — `npm run web:dev` from repo root; `npm run test:e2e` runs the Playwright suite in `web/e2e/`.
5. **License** — by contributing, you agree your contributions are under the same [MIT License](LICENSE) as the project.
