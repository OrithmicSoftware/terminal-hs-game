# Security

This project is a **fictional, educational terminal game**. It does not perform real attacks against networks or systems.

If you find a **dependency vulnerability** or **supply-chain** concern in how the repo is built or distributed, please [open a private security advisory](https://github.com/OrithmicSoftware/terminal-hs-game/security/advisories/new) on GitHub or report via the organization’s preferred channel.

Do not use this software to violate laws or third-party terms of service.
