Optional Fallout-style terminal UI sounds (selection, browsing, click)

Place these files next to this README:
  ui_select.wav  — moving highlight / list navigation (arrow keys in pager)
  ui_browse.wav  — focus change / tab between controls
  ui_click.wav   — confirm (Enter, Space, button activate)

If any file is missing, the browser build uses short Web Audio synth tones instead.

Reference for the *kind* of sounds (button selection, browsing, clicking):
  https://www.youtube.com/watch?v=d4TYCdWBgxs

Do not commit copyrighted game audio unless you have the rights. Rip or record
from your own licensed copy of the game, or use original samples.
