Field manual // operator protocol

Three moves. One window. Zero margin.

Every op runs on the same brutal loop: read the brief, build the lure, watch the trace. Miss a beat and the grid lights you up before the target ever clicks. Learn the rhythm or burn.

01 / READ THE BRIEF

Sable drops the dossier. Target company, known weak points, an alias that won't hold long. You have minutes to absorb it, because the trace clock starts the moment you open the uplink.

02 / BUILD THE LURE

Craft the phishing mail from a library of aliases, hooks and payloads. Every choice shifts the credibility bar. Too bland and the target ignores you. Too sharp and the filter flags it before dawn.

03 / DODGE THE TRACE

Each action bleeds trace. Reply to the mark's questions. Stall the security desk. Every message is a gamble — answer wrong and the bar slides red. Hit the exfil window before it closes or walk away with nothing.

Signal intercept // classified excerpt

The grid doesn't forgive. The handler doesn't wait.

Somewhere between the neon-bled server towers of Orion Corp and the encrypted back-channels Sable controls, there is a sliver of time where an operator can move clean — no signature, no flag, no name on any log. That window is exactly what you're paid to exploit.

Sable is a broker, not a friend. The jobs come in hot, the cut is fair, and the deniability is total — right up until it isn't. Orion runs a hardened perimeter, three rotating security desks, and an AI threat-tier system that upgrades your priority every time you breathe on the network. The margins are thin. The stories are electric.
